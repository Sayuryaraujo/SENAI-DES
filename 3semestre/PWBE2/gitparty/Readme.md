# stich-gitparty

Sistema web simples de gerenciamento de eventos.

O projeto permite visualizar eventos, realizar inscrições e acompanhar o status das vagas de forma simples e moderna.

# Funcionalidades do front

- Cadastro de usuários
- Login
- Lista de eventos
- Status do evento
- Inscrição em eventos
- Cancelamento de inscrição
- Lista de espera
- Interface moderna em tons de roxo

  # Funcionalidades do front

- Armazenar usuários cadastrados
- Validar login
- Gerenciar eventos
- Controlar número máximo de participantes
- Criar lista de espera automática
- Atualizar status das inscrições
- Salvar inscrições canceladas
- Integrar frontend com banco de dados

# Comando para o google stich
contendo isso:Crie um sistema web simples de gerenciamento de eventos da empresa GitParty.

O sistema deve ter: cadastro e login de usuários, lista de eventos, tela com detalhes do evento,inscrição e cancelamento de inscrição em eventos,limite máximo de participantes, lista de espera automática quando o evento lotar

* Status da inscrição:
  -Confirmada
  - Lista de espera
  - Cancelada

O estilo deve ser simples , usando tons de roxo e lilás, com aparência limpa e intuitiva.
As telas devem ser organizadas, fáceis de usar e focadas para gerenciadores de eventos.
Utilize cartões modernos, botões arredondados, sombras suaves e uma interface elegante com tema escuro em roxo.
